# Painter's Ref Lab M1 Reference Screenshots

## Purpose

These screenshots act as a visual regression baseline for Milestone 1 refactors.

## Recommended Capture Conditions

- same browser when possible
- same zoom level
- same theme, with Light and/or Dark noted
- use one representative JPG reference image
- medium viewport desktop size
- optional second mobile capture later
